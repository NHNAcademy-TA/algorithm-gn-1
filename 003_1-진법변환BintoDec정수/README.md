# 3. 진법 변환2 (2진수 -> 10진수)

**[ 문제 ]**
<br/>
문자열로 된 2진수들을 10진수 숫자로 변환

<br/>

**[ 조건 ]**

- Integer.xxx() 사용 금지

<br/>

**[ 예시 ]**

| 입력            | -> | 출력   |
|---------------|----|------|
| 101           | -> | 5    |
| 1011          | -> | 11   |
| 1011111       | -> | 95   |
| 1111111111111 | -> | 8191 |
